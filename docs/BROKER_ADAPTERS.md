# Broker Adapter Interface

Implement each broker behind a common interface:

```python
class BrokerAdapter:
    def get_accounts(self): ...
    def get_positions(self): ...
    def get_buying_power(self): ...
    def preview_order(self, order): ...
    def submit_order(self, signed_order): ...
    def cancel_order(self, order_id): ...
```

Supported first adapters should be paper-trading providers. Live adapters require explicit user approval, broker approval, and production audit logging.
